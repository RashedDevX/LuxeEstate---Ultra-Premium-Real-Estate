## ‚öôÔ∏è Getting Started

Follow these steps to set up and run LuxeEstate locally:

### Prerequisites
* Node.js (LTS version)
* npm or yarn
* A database (e.g., PostgreSQL, MongoDB)

### Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/RashedDevX/luxe-estate.git](https://github.com/RashedDevX/luxe-estate.git)
    cd luxe-estate
    ```

2.  **Install Dependencies:**
    ```bash
    npm install  # or yarn install
    ```

3.  **Environment Variables:**
    Create a `.env.local` file in the root directory and add your environment variables.
    ```env
    DATABASE_URL="your_database_connection_string"
    NEXT_PUBLIC_MAPBOX_TOKEN="your_mapbox_public_token"
    # Add other necessary keys like API secrets, etc.
    ```

4.  **Run the Development Server:**
    ```bash
    npm run dev
    ```
    Open your browser to `http://localhost:3000` to view the application.
